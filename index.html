Terminal From Here

Quickly open up an external terminal window from within NetBeans.

Download .zip Download .tar.gz View on GitHub

For some reason I like to use the Terminal (aka Command Prompt, if you're on Windows) a lot while I'm developing software. I used it to run scripts on my code, make commits and push them to GitHub and a lot of other stuff.

In the past I used to open up the folder that contains my NetBeans project and copy its location to terminal so I could 'cd' into that folder. Since I was doing this quite often, I figured it would be nice to add a feature for this to NetBeans and here it is: Terminal From Here.
